# ImageJ Particle Analysis

## Objective
To analyze and count particles using ImageJ.

## Method
- Converted image to 8-bit
- Applied threshold
- Converted to binary mask
- Used watershed to separate particles
- Used Analyze Particles to count and measure

## Results
Total particles counted: 24

## Parameters Used
- Size: 1000–Infinity
- Circularity: 0.80–1.00
- Exclude edges: Enabled

- ## Conclusion
ImageJ effectively detected and counted particles with high accuracy.
